📖 About
AI Car Tool is a Ruby on Rails application that leverages artificial intelligence to provide powerful automotive insights and analysis. Whether you're a car enthusiast, mechanic, or automotive professional, this tool helps you make informed decisions about vehicles with AI-driven recommendations and analysis.

🚀 Getting Started

### Prerequisites
```
ruby >= 3.x
rails >= 7.x
postgresql (installed and running)
node >= 18.x
yarn or npm
```

### Installation
```bash
# 1. Clone the repo
git clone https://github.com/sohamchavan07/Aicartool.git
cd Aicartool

# 2. Install Ruby dependencies
bundle install

# 3. Install JavaScript dependencies
yarn install
# or
npm install

# 4. Set up environment variables
cp .env.example .env

# 5. Set up the database
rails db:create db:migrate db:seed

# 6. Start the development server
bin/dev
```

Open http://localhost:3000 in your browser.

---

💡 Usage
```bash
# Generate a new model
rails generate scaffold Car brand:string model:string year:integer

# Run migrations
rails db:migrate

# Start the server
rails server

# Run tests
rails test
```
